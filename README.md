## 🖥️ What is conty-design-system?

Contry Design System is a collection of design tools and components. It helps you keep styles and elements uniform in your projects. These include buttons, colors, fonts, and other building blocks for websites or apps. This system is built so you can add or adjust parts quickly and smoothly.

Though it is built mainly for developers, anyone can download and run its tools on their Windows computer. You can explore the design components and see how they look or behave. If you want to use these tools, this guide will help you step-by-step.

## 📂 What’s inside the package?

Inside the extracted folder, you will find several parts:

- `@conty/design-system`: The main package for using the design system outside.
- `@conty/tokens`: Basic colors, fonts, and style tokens used throughout.
- `@conty/ui`: Reusable user interface components like buttons and inputs.
- `@conty/eslint-config`: Settings for code style checks, meant for developers.
- `@conty/tsconfig`: Configuration presets for TypeScript projects (for developers).
- `@conty/storybook`: A visual catalog that shows all UI components in one place.

For an average user, the most relevant parts will be `@conty/design-system`, `@conty/tokens`, and `@conty/ui`.

## 🛠️ How to use components in your projects (Optional)

If you want to use the design system in React projects, here is a simple way for users new to programming.

You need a project already using React, a common tool for building websites or apps. Follow these basic steps:

1. Open your command prompt.
2. Navigate to your project folder.
3. Run this command to install two main packages:

```
npm install @conty/ui @conty/tokens
```

4. Alternatively, to get the full entry package, run:

```
npm install @conty/design-system
```

5. In your React code, import styles and components:

```js
import "@conty/ui/styles.css";
import { Button } from "@conty/ui";

function Example() {
  return <Button variant="solid">Continue</Button>;
}
```

This will display a button styled by the design system.

If you do not use React or coding, you can skip this section and just explore the design system files.
