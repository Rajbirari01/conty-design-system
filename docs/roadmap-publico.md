# Roadmap Publico

Este arquivo mostra, de forma simples, no que estamos focados agora e como voce pode contribuir.

## Onde estamos focando

1. Base do Design System estavel (`tokens`, `ui` e pacote de entrada).
2. Qualidade continua (lint, typecheck, build e stories atualizadas).
3. Evolucao incremental de componentes sem quebrar quem ja usa.

## O que estamos construindo agora

- Melhorar componentes base (ex.: `Button`, `Badge`, `Card` e afins).
- Garantir consistencia visual com tokens.
- Tornar a documentacao mais clara para uso e contribuicao.

## Proximos passos

- Expandir componentes de formulario e layout.
- Melhorar cobertura de estados em stories (default, disabled, erro).
- Refinar experiencia de release e changelog.

## Boas contribuicoes para começar

- Corrigir texto e clareza dos docs.
- Adicionar exemplos de uso de componentes no Storybook.
- Ajustar estados faltantes em componentes existentes.
- Melhorar nomes de props, funcoes e arquivos para ficarem mais claros.

## Regra de ouro

Contribuicoes pequenas, frequentes e bem explicadas ajudam o projeto a crescer com consistencia.
